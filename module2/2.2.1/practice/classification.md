# Классификация CSV-файлов

| CSV-файл | Сущность | Тип | Обоснование |
|----------|----------|-----|-------------|
| products.csv | Товар | Справочник | Содержит статичные данные о товарах, используется как справочник |
| employees.csv | Сотрудник | Справочник | Содержит данные о сотрудниках |
| customers.csv | Покупатель | Справочник | Содержит данные о покупателях |
| shops.csv | Магазин | Справочник | Содержит данные о точках продаж |
| categories.csv | Категория | Справочник | Справочник категорий товаров |
| cities.csv | Город | Справочник | Справочник городов |
| countries.csv | Страна | Справочник | Справочник стран |
| sales.csv | Продажа | Транзакция/Факт | Содержит записи о каждой продаже, ссылается на справочники |
