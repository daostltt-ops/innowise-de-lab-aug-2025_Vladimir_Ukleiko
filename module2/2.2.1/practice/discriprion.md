# Предметная область: Магазин продуктов

## Описание
Система управляет продажами продуктового магазина. В ней хранятся данные о товарах, сотрудниках, покупателях, магазинах, продажах, а также справочная информация о категориях, городах и странах.

## Сущности и их назначение
| Сущность | Описание |
|----------|----------|
| products | Товары, которые продаются в магазине (фрукты, овощи и т.д.) |
| employees | Сотрудники, работающие в магазинах |
| customers | Покупатели, совершающие покупки |
| shops | Магазины (точки продаж) |
| sales | Записи о продажах (транзакции) |
| categories | Категории товаров (например, Fruits, Vegetables) |
| cities | Города, где расположены магазины |
| countries | Страны, где находятся города |

## Связи между сущностями
- products → categories (многие товары относятся к одной категории)
- sales → products (многие продажи ссылаются на один товар)
- sales → employees (многие продажи совершены одним сотрудником)
- sales → customers (многие продажи совершены одним покупателем)
- shops → cities (много магазинов в одном городе)
- cities → countries (много городов в одной стране)
