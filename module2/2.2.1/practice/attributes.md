# Анализ атомарности атрибутов

## 1. Сущность: Products (Товары)

### Список атрибутов:
1. product_id
2. product_name
3. price
4. category_id
5. class
6. modify_timestamp
7. resistant
8. is_allergic

### Анализ атомарности:

| Атрибут | Атомарен? | Обоснование |
|---------|-----------|-------------|
| product_id | Да | Числовой идентификатор, одно значение на товар |
| product_name | Да | Одно название товара в виде строки |
| price | Да | Числовое значение (цена), десятичная дробь |
| category_id | Да | Числовой идентификатор категории, ссылка на другую таблицу |
| class | Да | Одиночный символ (A, B или C) |
| modify_timestamp | Да | Одна дата и время в едином формате |
| resistant | Да | Булево значение (Yes/No) |
| is_allergic | Да | Булево значение (Yes/No) |

Вывод: все атрибуты атомарны, так как каждая ячейка содержит ровно одно неделимое значение.
